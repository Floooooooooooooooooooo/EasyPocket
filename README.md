# EasyPocket
A small gadget for mobile phones to have a storage for your cards and some money.

## Project Status
üöß **Prototype Phase** - First prototype (V1) specifications complete

## Overview
EasyPocket is a compact, phone-attachable accessory designed to store:
- 2-4 credit/debit cards
- Folded cash (up to 10 bills)
- Attaches securely to the back of your phone or phone case

Perfect for minimalists who want to carry less while keeping essentials accessible.

### Getting Started

#### Option 1: 3D Print at Home
```bash
1. Review CAD_GUIDE.md to create 3D model
2. Export as STL file
3. Slice with appropriate settings (see TECHNICAL_SPECS.md)
4. Print with TPU filament
5. Follow assembly instructions in PROTOTYPE_V1.md
```

#### Option 2: Hand-Build Prototype
```bash
1. Purchase materials from MATERIALS.md
2. Follow pattern specifications in TECHNICAL_SPECS.md
3. Cut and assemble per PROTOTYPE_V1.md instructions
4. Test according to TESTING_GUIDE.md
```

#### Option 3: Order from Service
```bash
1. Create 3D model following CAD_GUIDE.md
2. Export STL file
3. Upload to service (Shapeways, Sculpteo, etc.)
4. Select TPU material
5. Receive and test
```
